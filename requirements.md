# Sprint Requirements

## gh-toy-yjy: Note Functionality
Implement a basic note-taking feature where users can add and view notes for resources.
AC: Users can add notes to resources. Notes are stored and can be retrieved.

## gh-toy-06i: Pagination
Implement pagination for the resource listing API and CLI to improve performance with large datasets.
AC: The list command/API supports limit and offset parameters.

## gh-toy-gw1: Full-text Search
Implement a full-text search capability to allow users to find resources by keywords in titles and descriptions.
AC: A search function returns resources matching a provided query string.
