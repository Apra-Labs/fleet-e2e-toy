# Requirements

We need to implement the following three P1 tasks:

1. **Implement a help command** (`gh-toy-kbk`)
   - Add a command or option to print usage information and help instructions.

2. **Add input validation for empty or blank strings** (`gh-toy-v6z`)
   - Validate inputs and return appropriate errors/warnings when empty or blank strings are provided.

3. **Add --version flag to CLI** (`gh-toy-4ef`)
   - Add a `--version` flag to print the current version of the CLI tool.
